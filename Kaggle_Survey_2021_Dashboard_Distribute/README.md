# 2021 Kaggle Survey 대시보드 배포

![](img/Kaggle_survey_2021_Dashboard_Distribute.png) 

## 목표 


 - 2021 Kaggle Survey 데이터를 바탕으로 시각화, 피벗테이블 등으로 대시보드를 제작 및 배포한다.

## 설명


- 사용언어 : Python (3.9.0)


- 사용 라이브러리
  - Plotly (5.4.0)
  - Dash (1.21.0)
  - Numpy (1.21.4)
  - Pandas (1.3.5)
  - Gunicorn (20.1.0)


- 사용 에디터 : PyCharm Community Edition (2021.2.3)
